Whats this

This is meant to be a github-like activity calendar where we can view a user's actions by days. Here i unite my activity over some computer science related websites that i use frequently (or every now and then).

How this works

All the necessary data is gathered via webscraping, but since i'm not up to pay for a website hosting, i'm using github pages which is free, although comes with some limitations such as being a static page. A static page is quite a big problem to keep this calendar up to date, so my work around was to output my webscraping to a json, which is put into this page's javascript. This way, all i have to do is to run my python webscraper every a certain amount of time and commit the changed json to this repository, which is a task that i can put my computer to automatically execute once it is turned on. I choose to update every month so that the repository don't get polluted by automatic commits.
The webscraper is also in this page's repository if you want to check it out.
