How this works

Getting all the data and updating this page

  • All the necessary data is gathered via Web Scraping.
  • Since i chose to use github pages, it means that i have no backend service to send a HTTP request to get the data from my Web Scraping.
  • My solution was to put the data in a json, inside this page's JavaScript file.
  • Then all i have to do is run my Web Scraper, put the data in a json and then put it in a JavaScript file (plain text), all of which is easily done with python.
  • Not the greatest solution ever, because it's prone to bugs (my python program assumes that the json is in the first line of the JavaScript file), but it's a "good enough" solution for this simple project.
2026 update: my original idea was to set my personal computer, once it was turned on, to run the Web Scraper and automatically update this page's data. And it worked... for like 2 months, until I messed with configuration files and didn't bother fixing this automation.
Conclusion: it works, but it needed a dedicated computer for all the automation. But if I had that, I might aswell just had a backend server and defeat the whole purpose of this "whacky" implementation.

The whole visual part of this page was done entirely with raw javascript and css.
